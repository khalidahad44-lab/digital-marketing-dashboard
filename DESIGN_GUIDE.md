# DESIGN GUIDE

## Color Palette
- **Primary Color:** #4A90E2  
- **Secondary Color:** #50E3C2  
- **Accent Color:** #F5A623  
- **Background Color:** #FFFFFF  
- **Text Color:** #4A4A4A

## Typography
- **Header Font:**  
  - Family: 'Roboto', sans-serif  
  - Weight: Bold  
  - Size: 24px

- **Body Font:**  
  - Family: 'Open Sans', sans-serif  
  - Weight: Regular  
  - Size: 14px

## Layout Specifications
- **Grid System:**  
  - 12-column grid layout  
  - Gutter width: 16px  
  - Margin: 20px

- **Container Width:**  
  - Max width: 1200px  
  - Mobile responsive: yes

## Styling Instructions
- **Buttons:**  
  - Background color: #4A90E2  
  - Text color: #FFFFFF  
  - Border radius: 4px  
  - Padding: 10px 20px  
  - Transition: background-color 0.3s ease

- **Links:**  
  - Color: #4A90E2  
  - Hover color: #F5A623

## Accessibility Considerations
- **Keyboard Navigation:** Ensure all interactive elements are accessible via keyboard.  
- **Color Contrast:** Maintain a contrast ratio of at least 4.5:1 for text.  
- **Alt Text for Images:** Provide descriptive alt text for all images.

This document provides a comprehensive guide for designers and developers to maintain consistency throughout the digital marketing dashboard project.
