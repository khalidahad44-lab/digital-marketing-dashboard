# Quick Start Guide for Google Sheets Dashboard

## 5-Minute Setup Steps
1. **Clone the Repository**: `git clone https://github.com/khalidahad44-lab/digital-marketing-dashboard.git`
2. **Navigate to the Project Directory**: `cd digital-marketing-dashboard`
3. **Install Dependencies**: Use `npm install` to install any necessary packages.
4. **Set Up Google Sheets**: Follow the configuration instructions in the documentation to link your Google Sheets.
5. **Run the Application**: Start the dashboard using `npm start` and open your browser to see the dashboard in action!

## Troubleshooting
- **Error: Unable to connect to Google Sheets**: Make sure you have set the correct API credentials and enabled the necessary APIs in your Google Cloud Console.
- **Missing Packages**: Run `npm install` again to ensure all dependencies are installed.
- **Dashboard Not Loading**: Check the console for errors and ensure that all environment variables are correctly set.

## Pro Tips for Building the Dashboard
- Use named ranges in Google Sheets for better data management and easier reference.
- Regularly update your dependencies to benefit from the latest features and security fixes.
- Take advantage of Google Sheets functions to preprocess your data before it reaches the dashboard.
